Projeto WebScrapering

Objetivo:

Desenvolver uma aplicação em Python para realizar Web Scraping, permitindo a coleta automática de dados em sites da internet, como nomes de produtos, preços, descrições e outras informações relevantes.

Os dados extraídos serão processados, organizados e armazenados em planilhas Excel, facilitando análises, comparações e a geração de relatórios. O projeto também tem como objetivo aplicar boas práticas de programação, manipulação de dados e automação de processos, utilizando bibliotecas como Requests, BeautifulSoup e Pandas.

Funcionalidades previstas
Coletar dados de páginas da web.
Extrair informações específicas de produtos ou serviços.
Organizar os dados em tabelas.
Exportar os resultados para arquivos Excel (.xlsx).
Permitir futuras integrações com bancos de dados e dashboards para análise das informações.
